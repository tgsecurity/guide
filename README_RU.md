# Базовая конфиденциальность

- Номер телефона
  - Кто видит мой номер телефона — **Никто**
  - Кто может найти меня по номеру — **Мои контакты**
- Последняя активность
  - Кто видит время последнего входа — **Никто**
- Пересылка сообщений
  - Кто может ссылаться на мой аккаунт при пересылке сообщений — **Никто**
- Фотография профиля
  - Кто видит мою фотографию — **Мои контакты**
- Звонки
  - Кто может звонить мне — **Никто** ⚠️ Разрешайте, добавляя в исключения
  - Peer-to-peer — **Не использовать** ⚠️ На старых версиях возможна утечка IP-адреса
